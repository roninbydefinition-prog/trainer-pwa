# Presence & Flirting Trainer

A browser-based, localStorage-powered app for practicing social presence and flirting drills.

- Create and log field drills and submissions
- Get random reminders for surprise practice
- Grade and review your past entries
- Export your log as CSV
- All data stays local in your browser

## Usage

1. Open `index.html` in your browser.
2. Allow notifications if prompted (for reminders).
3. Create drills and submit practice sessions.
4. Export your data as CSV to back up or share.

No server or build step required.

---

> **Tip**: The app is self-contained as a single HTML file. Just open or deploy as a static page.
